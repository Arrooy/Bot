-[3.7V 600mAh Lipo Battery](http://www.dx.com/p/3-7v-600mah-lipo-battery-1-to-4-charger-for-syma-x5c-1-x5sw-drone-jjrc-h5c-v943-931-416020#.WVoJeYg1-Uk).
